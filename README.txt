# PROG-Miniproject-V1M-5

This is the code for a game using the marvel API
It consists out of the player guessing which super hero is being described being able to ask for hints in the process.



First install the following packages

>pip install requests
>pip install pillow

Run the game by executing in python 3 the tinkergui.py file

>python tinkergui.py

or

>python3 tinkergui.py
